# 🎫 Ticket Management System

## 📌 Overview

This project is a REST API built with Spring Boot for managing support tickets. It allows creating and listing tickets, forming the base for a complete CRUD system.

## 🚀 Technologies

* Java 17+
* Spring Boot
* Maven
* H2 Database (in-memory)
* JPA / Hibernate

## 📂 Project Structure

```
src/
 └── main/
     ├── java/com/michelle/tickets/
     │    ├── controller/
     │    ├── model/
     │    ├── service/
     │    └── TicketsApplication.java
     └── resources/
```
## 🧠 Future Improvements

* Full CRUD (update, delete, find by id)
* Integration with PostgreSQL
* Input validation
* DTO pattern
* Authentication (Spring Security)
