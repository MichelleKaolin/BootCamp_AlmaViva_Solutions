# 🚀 Java OOP Bootcamp Project | Projeto POO em Java

## 📌 About the Project | Sobre o Projeto

**EN:**
This project was developed as part of the Digital Innovation One (DIO) challenge to practice Object-Oriented Programming (OOP) concepts in Java. It simulates a bootcamp platform where developers can enroll in courses and mentorships, progress through content, and accumulate experience points (XP).

**PT-BR:**
Este projeto foi desenvolvido como parte do desafio da Digital Innovation One (DIO) para praticar conceitos de Programação Orientada a Objetos (POO) em Java. Ele simula uma plataforma de bootcamp onde desenvolvedores podem se inscrever em cursos e mentorias, progredir nos conteúdos e acumular experiência (XP).

---

## 🧠 Concepts Applied | Conceitos Aplicados

* ✔️ Abstraction (Abstração)
* ✔️ Encapsulation (Encapsulamento)
* ✔️ Inheritance (Herança)
* ✔️ Polymorphism (Polimorfismo)
* ✔️ Collections (Set, HashSet)
* ✔️ Java Streams API

## ⚙️ How It Works | Como Funciona

**EN:**

* A Bootcamp contains multiple learning contents (courses and mentorships).
* Developers (Dev) can enroll in a bootcamp.
* As they progress, they complete content and earn XP.
* XP is calculated using polymorphism based on content type.

**PT-BR:**

* Um Bootcamp contém vários conteúdos (cursos e mentorias).
* Desenvolvedores podem se inscrever no bootcamp.
* Ao progredirem, concluem conteúdos e acumulam XP.
* O cálculo de XP utiliza polimorfismo conforme o tipo de conteúdo.
